### README.md

# JPetStore Gatling Load Test

Проект нагрузочного тестирования демо-приложения JPetStore.

## Стек

- Java 17
- Maven
- Gatling
- Gatling Java

## Стенд

https://petstore.octoperf.com

## Описание

В проекте реализованы нагрузочные сценарии для интернет-магазина JPetStore.

Покрыты основные пользовательские действия:

- открытие магазина;
- просмотр каталога;
- поиск товара;
- добавление товара в корзину;
- авторизация пользователя;
- действия авторизованного пользователя с каталогом и корзиной.

Тестирование выполняется по black-box подходу, без доступа к серверу, логам и базе данных.

## Структура проекта

Проект разделён на несколько логических частей:

- `config` — конфигурация тестов;
- `feeders` — тестовые данные;
- `steps` — HTTP-шаги;
- `scenarios` — пользовательские сценарии;
- `simulations` — профили нагрузки.

## Реализованные сценарии

- `Open Store Scenario` — открытие магазина.
- `Browse Catalog Scenario` — просмотр каталога и карточки товара.
- `Search Product Scenario` — поиск товара.
- `Add Item To Cart Scenario` — добавление товара в корзину.
- `Authorized User Scenario` — авторизация, просмотр товара, добавление в корзину и выход.

## Профили нагрузки

В проекте реализованы следующие запуски:

- `ValidationSimulation` — проверка корректности сценариев.
- `SmokeSimulation` — быстрый smoke test.
- `LoadSimulation` — основной нагрузочный тест.
- `StepCapacitySimulation` — ступенчатое увеличение нагрузки.
- `ReliabilitySimulation` — тест стабильности.

## Авторизация

Для сценария авторизованного пользователя используется заранее созданный тестовый аккаунт.

Регистрация нового пользователя не входит в нагрузочный профиль.

## Отчёты

После выполнения тестов Gatling формирует HTML-отчёты в директории `target/gatling`.

## Примечание

JPetStore — публичный демонстрационный стенд, поэтому нагрузка в проекте ограничена. 
Цель проекта — показать методику нагрузочного тестирования, реализацию сценариев и анализ поведения приложения при безопасной учебной нагрузке.
